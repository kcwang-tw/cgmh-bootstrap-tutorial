# 活用Bootstrap套版打造友善網頁

2016年6月長庚醫院醫療資訊管理部組會技術分享

## 使用說明

### Demo範本
- 請使用Visual Studio 2013以上版本開啟
- 範本包含純HTML、Web Froms與ASP.Net MVC三種版本
- 內含基本應用demo code與實際呈現範例
- 如欲修改可直接下載或自行於GitHub Fork到自己的帳號
